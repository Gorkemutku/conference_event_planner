# Paradise Nursery - Plant Shopping Application

A modern React-based e-commerce application for browsing and purchasing indoor plants. Paradise Nursery allows users to explore plants by categories, add items to their shopping cart, and manage their purchases with an intuitive interface.

## Features

- **Home Page**: Welcome page with background image, company information, and "Get Started" button
- **Product Listing Page**: Browse 6+ unique indoor plants organized into 3+ categories with thumbnails, names, and prices
- **Shopping Cart**: View all items, total quantities, total cost, and manage quantities with add/remove buttons
- **Navbar**: Dynamic cart counter, navigation between pages (Home, Products, Cart)
- **Redux State Management**: Cart state managed using Redux for predictable state updates
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technologies Used

- React 18
- Redux Toolkit
- Vite (build tool)
- CSS3

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development: `npm run dev`
4. Build for production: `npm run build`
